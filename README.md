# Package Details

reqwest - this package used for external API call
serde_json - to handle JSON object 
serde - Serialize, Deserialize
axum - works as express in nodejs
tower-http - Adds features like request logging, tracing, compression, CORS, authentication, etc.
tokio - An asynchronous runtime for Rust. Provides event loop, task scheduling, async I/O.
lambda_http - Adapter for running HTTP services inside AWS Lambda.
